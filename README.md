# STEM Engineering Summer Academy - Security Education Modules

Interactive cybersecurity education activities for high school students, combining hands-on coding with real-world security concepts using p5.js.

## Module Structure

```
oss-slu/cybersecurity_activities/
├── README.md (this file)
├── CorrectHorseBatteryStapler/
│   └── README.md
└── ShiftLeft/
    └── README.md
```

## Overall Objectives

Students will gain practical understanding of password security, software supply chain vulnerabilities, and security visualization techniques through hands-on coding activities.

## Available Modules

### Correct Horse Battery Stapler

60-minute interactive p5.js application demonstrating password security concepts through real-time entropy calculation, visual crack time estimation, and particle system animations. Based on XKCD 936 comic principles.

### Shift Left

60-minute hands-on exploration of dependency vulnerabilities using an interactive package.json scanner, real CVE database integration, and dependency tree visualization to understand supply chain security risks.

## Getting Started

Each module contains all necessary files in its directory. Navigate to the desired module directory and follow the README instructions for that specific activity.

## Technical Stack

Built with p5.js for visualizations, vanilla JavaScript for security logic, and HTML5/CSS3 for interactive interfaces. Some modules integrate real security APIs and databases.
